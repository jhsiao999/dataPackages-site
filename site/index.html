Background

This site list some single-cell RNA-seq data sets downloaded from GEO. We store each of the data sets in ExpressionSet object and make the data available as an R package on GitHub.

  • Codes for making ExpressionSet objects

Data packages

  • Human
    • Tung et al., 2016: Human iPSC, counts, UMI
    • Leng et al., 2015: Human ESC, reads, cell-cycle sorted
  • Mouse
    • Buettner et al., 2015: mouse ESC, reads, ERCC spike-in control
    • Jaitin et at., 2014: mouse spleen cells, counts, UMI
    • Treutlein et al., 2014: mouse lung cells, reads
    • Zeisel et al., 2015: Mouse brain, counts + UMI
